# VAA Global Business School for Artisans

Static site — drop the folder onto Vercel, Netlify or any host.

## Before going live
Open `assets/js/main.js` and set:
1. `VAA.PAYSTACK_PUBLIC_KEY` — your live `pk_live_...` key
2. `VAA.APPS_SCRIPT_URL` — your Google Apps Script endpoint

Blog pages in `/blog/` are stubs — write the articles and deploy.
